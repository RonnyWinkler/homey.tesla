Enregistrez les données de votre véhicule, de votre climatisation avant de conduire ou optimisez la recharge...
D'innombrables façons de rendre votre Tesla encore plus intelligente et de l'intégrer dans votre maison intelligente.

Configuration :
Ajoutez votre véhicule en tant qu'appareil. Connectez-vous avec votre compte Tesla.
Une fois ajouté, vous avez la possibilité d’enregistrer l’application dans votre véhicule afin d’envoyer des commandes.
Vous pouvez ensuite ajouter d’autres appareils pour contrôler certaines fonctions du véhicule (climatisation, recharge…).

Pour les appareils énergétiques, ajoutez d’abord l’appareil EnergySite. Ensuite, ajoutez les appareils dépendants tels que les batteries et les panneaux solaires.

Remarques sur l’API :
Cette application utilise l’API Tesla Fleet, qui prend en charge le chiffrement de bout en bout des commandes envoyées au véhicule.
Pour utiliser cette application, vous devez disposer d’un compte développeur Tesla. La documentation est disponible dans l’écran d’association.
Pour envoyer des commandes au véhicule, l’application doit être enregistrée dans le véhicule. Vous pouvez le faire lors de l’ajout du véhicule ou ultérieurement via le dialogue de réparation.
L’ajout en tant que clé de véhicule peut être restreint si vous n’êtes pas le propriétaire et que le véhicule est enregistré au nom du loueur.
