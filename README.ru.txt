Зарегистрируйте данные вашего автомобиля, кондиционер перед поездкой или оптимизируйте зарядку...
Бесчисленные способы сделать вашу Tesla еще умнее и интегрировать ее в свой умный дом.

Настраивать:
Добавьте свой автомобиль в качестве устройства. Войдите в свою учетную запись Tesla.
После добавления у вас есть возможность зарегистрировать приложение в своем автомобиле, чтобы отправлять команды на автомобиль.

Затем вы можете добавить дополнительные устройства для управления конкретными функциями автомобиля (климат, зарядка...).

Примечания API:
Это приложение использует API Tesla Fleet. Это поддерживает сквозное шифрование команд, отправляемых на транспортное средство.
Чтобы отправлять команды в автомобиль, приложение должно быть зарегистрировано в автомобиле. Для этого воспользуйтесь вызовом при добавлении автомобиля или выполните этот шаг позже в диалоге ремонта.
Добавление в качестве ключа от автомобиля может быть ограничено, если вы не являетесь владельцем, но автомобиль зарегистрирован на арендодателя.

Ограничения по ставкам API:
Tesla ввела ограничения по сети:
- 300 считываний данных в день
- ~50 команд в день
- менее 50 команд запуска/остановки зарядки или команды пробуждения в день
