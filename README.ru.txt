Зарегистрируйте данные вашего автомобиля, кондиционер перед поездкой или оптимизируйте зарядку...
Бесчисленные способы сделать вашу Tesla еще умнее и интегрировать ее в свой умный дом.

Настройка:
Добавьте свой автомобиль как устройство. Войдите в свою учетную запись Tesla.
После добавления у вас будет возможность зарегистрировать приложение в автомобиле для отправки команд.
Затем вы можете добавить дополнительные устройства для управления конкретными функциями автомобиля (климат, зарядка и т.д.).

Для энергетических устройств сначала добавьте устройство EnergySite. Затем добавьте зависимые устройства, такие как аккумуляторы и солнечные панели.

Примечания к API:
Это приложение использует Tesla Fleet API, которое поддерживает сквозное шифрование команд, отправляемых в автомобиль.
Для использования приложения необходима учетная запись разработчика Tesla. Документация доступна в режиме сопряжения.
Чтобы отправлять команды в автомобиль, приложение должно быть зарегистрировано в автомобиле. Для этого используйте вызов при добавлении автомобиля или выполните этот шаг позже в диалоговом окне ремонта.
Добавление в качестве ключа автомобиля может быть ограничено, если вы не являетесь владельцем, а автомобиль зарегистрирован на лизингодателя.
